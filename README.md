# Steps to start project


# Start with git
`git clone https://github.com/dhavalrajani92/Task-NTS-UserData.git`

`git fetch`

`git checkout master`
# Setup npm module
`npm install`

Please use node 8.x version

# Enviroment variable setup

# For application
`export PORT=4002`

## Starting application developement
`npm run dev`

## Build application
`npm run build && npm start`
